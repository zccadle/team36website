Data visualisation on The Red Cross' GO Dashboard

Abstract

Problem statement

The IFRC GO platform is designed to integrate iframe snippets from BI software like PowerBI and Tableau onto dedicated tabs for emergency, regional, and country pages. However, while these dashboards facilitate rapid data discovery, they often lack the capacity for in-depth analysis and elegant visualization.

Team 36's solution

Team 36 has developed the Chart Gallery, a comprehensive solution that showcases advanced data visualizations such as force-directed graphs, heatmaps, bubble charts, chord diagrams, and more. Leveraging responsive data visualization tools like D3.js, chart.js, ESRI, and others, this project demonstrates and documents enhanced data visualization capabilities within IFRC GO. The focus lies on visualizing ongoing emergency operations, programs, and country-level data within the IFRC network.

Achievement and impact

With the implementation of the Chart Gallery, every graph now boasts vibrant and highly interactive features. Each graph offers zooming and panning functionalities, along with tooltips for displaying additional information, significantly enhancing the user experience. Moreover, these example visualizations serve as templates, enabling even users with minimal knowledge of React or other JavaScript frameworks to easily replicate and customize their own graphs. This initiative not only improves data analysis capabilities but also empowers users to create compelling visualizations tailored to their specific needs, ultimately driving greater engagement and understanding across the IFRC community.
