Abstract

IFRC GO is built to embed iframe snippets using BI software on dedicated tabs for emergency, regional and country pages. While such dashboards provide rapid data discovery, Power BI makes it difficult for users to set up and update emergency statuses, and shows limited data visualisation options.

Using responsive data visualisation tools such as chart.js, D3.js, ESRI, and React, we have enhanced the data visualization experience for users by making a chart type gallery to display the data. Our work also allows us to add flexible grids and layouts to achieve mobile friendliness.

Features
Feature 1
Chart Gallery


You want to look for data about a recent emergency event? You can now use our chart gallery to find the data you are looking for and display it on a dashboard of your choice.

Feature 2
Documentation


You can now reproduce charts of your own with the help of our documentation, where we explained how to used our technologies.

Feature 3
Mobile Friendly


In order to make our chart gallery more accessible, we have made it usable on mobile devices, ipads and many other gadgets.
